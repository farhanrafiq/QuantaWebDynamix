SITEMAP ARCHITECTURE
- Create a sitemap index /public/sitemap.xml that links to:
  • sitemap-pages.xml
  • sitemap-posts.xml
  • sitemap-categories.xml
  • sitemap-tags.xml
  • sitemap-apps.xml (your SaaS and One-Time apps)
  • sitemap-images.xml
- Each child uses <url><loc><lastmod><changefreq><priority></priority></changefreq></lastmod></loc></url>
- Split automatically at 45k URLs per file if needed and number parts.
- Gzip every sitemap and serve .xml and .xml.gz.
- Add hreflang alternates if locales > 1.
- Update lastmod from filesystem or content frontmatter.
- Regenerate sitemaps on build and expose a script npm run seo:build.
